Welcome to Bunnify

This application manages URL bookmarks and provides quick redirects.

How to Use

1. Simple Bookmarks

Navigate to /<key>/ to be redirected to the bookmark URL.

Example:

  • /c/ → Redirects to Google Calendar
  • /vault/ → Redirects to Vault

2. Parameterized Bookmarks

Some bookmarks require parameters. You can pass them via query parameters or the search interface.

Query Parameter Examples:

  • /pr/?pr_number=12345 → Redirects to PR #12345
  • /g/?search_terms=django+tutorial → Redirects to Google search

Search Interface (Recommended):

  • /search/?q=pr 12345 → Uses default repo
  • /search/?q=pr 12345 Shopify/shopify-build → Overrides default repo
  • /search/?q=g django tutorial → Google search

Bookmarks with multiple parameters support defaults. Required parameters come first, optional parameters (with defaults) can be overridden.

3. View All Bookmarks

Visit /list/ to see all available bookmarks.

Setup Instructions

  1. Run migrations: python manage.py migrate
  2. Load bookmarks: python manage.py load_bookmarks
  3. Start server: python manage.py runserver
